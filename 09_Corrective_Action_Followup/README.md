# Stage 09 - Follow-Up on Corrective Actions

This stage closes the audit cycle. For each nonconformity, the auditee prepares a corrective action
plan, carries it out, and provides evidence. The auditor then verifies that the action was completed
and is effective before the finding is closed. Completing this loop is what demonstrates continual
improvement.

## Contents

| File | Purpose |
|---|---|
| `Corrective_Action_Plan.xlsx` | Two tabs: Corrective Action Plan (the three nonconformities with root cause, action, owner, target date, evidence, auditor verification, effectiveness, and closure) and OFI Follow-Up (status of the two opportunities for improvement). |

## What this stage records (per the audit requirement)

- The auditee's corrective action plan: root cause and the action taken for each nonconformity.
- The auditor's verification that each action was completed and is effective.
- The evidence collected to support closure.
- The closure status and date for each finding.
