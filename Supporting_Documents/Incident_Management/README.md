# Incident Management

This folder holds the supporting record used for handling evidence during an information security
incident or investigation.

| File | Purpose |
|---|---|
| `Chain_of_Custody_Form.docx` | A template for recording the handling of evidence collected during an incident, who collected it, when, where it was stored, and each transfer, so an unbroken, documented trail can be shown. |

The form supports ISO/IEC 27001:2022 Annex A control 5.28 (Collection of evidence) and the
organisation's incident response process. It is provided as a blank template to be completed if and
when an incident requires evidence to be preserved.

> Note: `.docx` files do not preview inline on GitHub. Download the form to view it in full.
