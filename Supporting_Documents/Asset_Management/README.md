# Asset Management

The asset register for Stark Industries Inc. is maintained in the combined workbook, not as a
separate file in this folder.

See: `Supporting_Documents/Statement_of_Applicability/Stark_Industries_Assets_Risk_Assessment_SoA.xlsx`

- The **Asset Register** tab holds the information assets, owners, classification, and supporting
  details.

A single combined workbook is used so there is one source of truth for assets, risks, and the SoA.
