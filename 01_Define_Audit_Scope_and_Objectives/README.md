# Stage 01 - Define Audit Scope and Objectives

This is the foundation stage of the internal audit. Before any fieldwork can begin, the audit
needs a clear boundary (what is being audited), a clear purpose (why), and a clear set of
yardsticks (what the audit measures against). Everything in the later stages traces back to the
decisions made here.

## What this stage establishes

- **The boundary** of the audit, taken from the organisation's defined ISMS scope.
- **The objectives** the audit is intended to achieve.
- **The criteria** against which audit evidence will be compared to decide conformity or
  nonconformity.

## How it feeds the rest of the audit

The scope sets what the audit plan (Stage 02) must cover. The criteria become the basis for the
clause-by-clause and control checklists (Stage 03). During fieldwork (Stage 05), evidence is
compared against these criteria, and any gaps become findings (Stage 06).
